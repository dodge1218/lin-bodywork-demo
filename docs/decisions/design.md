# Design Rationale

- Tone: modern, calm, premium-light (distinct from dark themes).
- Palette: white + slate neutrals + sky accent (60/30/10 distribution).
- Typography: system sans stack with strong weight contrast.
- Components: rounded-xl/2xl cards, soft borders, restrained shadows.
- Motion: subtle transitions on interactive controls; no flashy effects.
- Responsiveness: single-column mobile flows, multi-column desktop at md+.
